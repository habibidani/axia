# Environment Setup Guide

## Required Environment Variables

Add these to your `.env` file:

```env
# Application
APP_NAME=Axia
APP_ENV=local
APP_KEY=base64:... # Generated by php artisan key:generate
APP_DEBUG=true
APP_URL=http://localhost

# Database
DB_CONNECTION=sqlite
# For SQLite, create an empty database file:
# touch database/database.sqlite

# OpenAI API (REQUIRED for task analysis)
OPENAI_API_KEY=sk-your-api-key-here
OPENAI_MODEL=gpt-4-turbo-preview
# Alternative: gpt-3.5-turbo for faster/cheaper analysis

# Session & Cache
SESSION_DRIVER=file
CACHE_DRIVER=file
QUEUE_CONNECTION=sync

# Mail (optional, for password resets)
MAIL_MAILER=log
MAIL_FROM_ADDRESS="hello@axia.app"
MAIL_FROM_NAME="${APP_NAME}"
```

## Getting an OpenAI API Key

1. Visit https://platform.openai.com/api-keys
2. Sign up or log in
3. Create a new API key
4. Copy and paste it into your `.env` file

**Note**: OpenAI API usage requires billing setup. GPT-4 costs more but provides better analysis quality.

## First Run Setup

```bash
# 1. Copy environment file
cp .env.example .env

# 2. Generate application key
php artisan key:generate

# 3. Create SQLite database
touch database/database.sqlite

# 4. Add your OpenAI API key to .env
# Edit .env and set OPENAI_API_KEY=sk-...

# 5. Run migrations
php artisan migrate

# 6. Build frontend assets
npm run build

# 7. Start the server
php artisan serve
```

Visit http://localhost:8000 and you're ready to go!
